Python SDK для
дизайн-системы 1409

Подключай UI-компоненты, лого, фоны и шрифты в любой Python-проект.
Один импорт — и дизайн 1409 у тебя в коде.

Возможности

Всё что нужно для работы с 1409

🔗

HTML-теги одной строкой

Получи готовый <link> или <img> тег для любого компонента без лишнего кода.

⚡

Запросы к API

Все запросы к api.my1409.ru обёрнуты в удобный клиент. Авторизация не нужна.

📦

Экспорт в проект

Скачай любой компонент в папку проекта одним методом — pull().

🔤

Шрифты системы

Montserrat, Roboto, DejaVu — шрифты дизайн-системы готовы к копированию в проект.

Установка

Начало работы

terminal 
pip install 1409sdk
quickstart.py 
from 1409sdk.client import Client

client = Client()

# Получить HTML-тег для вставки на страницу
print(client.link_tag("logo"))
# <img src="https://api.my1409.ru/design?component=logo" alt="logo">

print(client.link_tag("button"))
# <link rel="stylesheet" href="https://api.my1409.ru/design?component=button">

# Вставить в шаблон
logo = client.link_tag("logo")
html = f"<html><body>{logo}</body></html>"

# Скачать компонент в папку
client.pull("logo", dest="./assets/")

# Скопировать шрифты
client.fonts(dest="./assets/fonts/")

Справочник

Методы Client

Метод
Описание
link_tag(component)
Возвращает готовый HTML-тег (<img>, <link>) для вставки компонента на страницу
get(component)
Запрашивает компонент с сервера, возвращает bytes
get_text(component)
Запрашивает компонент, возвращает строку (HTML, CSS и др.)
pull(component, dest)
Скачивает файл компонента и сохраняет в указанную папку
list()
Возвращает список доступных компонентов с сервера
fonts(dest)
Копирует шрифты дизайн-системы в папку проекта

Примеры

Интеграция в Flask

app.py 
from flask import Flask, render_template_string
from 1409sdk.client import Client

app = Flask(__name__)
client = Client()

@app.route("/")
def index():
    logo = client.link_tag("logo")
    bg   = client.link_tag("background")
    return render_template_string(f"""
        <html>
          <body>
            {bg}
            {logo}
          </body>
        </html>
    """)
